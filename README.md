# QuickList

QuickList is a streamlined shopping list management application built with the
 TALL stack (Tailwind CSS, Alpine.js, Laravel, and Livewire). It helps you
 organize your grocery shopping with categorized lists, product management, and
 a dashboard for quick access to your latest list.

## Development

You can use the provided `makefile` for common tasks:

- `make clear`: Clear caches.
- `make fmt`: Format code using PHP CS Fixer.
- `make test`: Run Pest tests.
- `make build`: Build assets for production.

## Testing

QuickList uses Pest PHP for feature and unit testing.

```bash
make test
```
